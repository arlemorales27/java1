Algoritmos y Lógica Computacional

Una guía completa para entender los fundamentos de la programación y el diseño de algoritmos

1. ¿Qué es el software?

El software es un conjunto de instrucciones o programas que le indican a una computadora qué hacer. Es el componente intangible de un sistema informático, a diferencia del hardware, que es el componente físico.

2. Tipos de software

Software Web

Aplicaciones que se ejecutan en un navegador, como Google Chrome o Firefox. Ejemplo: Gmail.

Software de Escritorio

Programas instalados en una computadora, como Microsoft Word o Visual Studio Code.

Software Móvil

Aplicaciones diseñadas para dispositivos móviles, como WhatsApp o juegos en Android/iOS.

3. Lenguajes de programación

Un lenguaje de programación es un medio para comunicarse con las computadoras mediante instrucciones. Los lenguajes más populares dependen del objetivo:

  • Web: Java, HTML, CSS
  • Móvil: Swift (iOS), Kotlin (Android)
  • Escritorio: C#, Java
  • Datos/IA: Python, R

4. ¿Qué es un algoritmo?

Un algoritmo es una secuencia de pasos finitos y ordenados para resolver un problema. Ejemplo: una receta de cocina es un algoritmo.

5. Tipos de algoritmos y ejemplos

  • Búsqueda: Búsqueda binaria para encontrar un elemento en una lista ordenada.
  • Ordenamiento: Algoritmo de burbuja para ordenar una lista.
  • Recursivos: Cálculo del factorial de un número.

6. Representación de un algoritmo

Pseudocódigo

Inicio
    Leer número
    Si número > 0 entonces
        Escribir "Número positivo"
    Sino
        Escribir "Número no positivo"
Fin

Diagrama de flujo

Un diagrama de flujo usa formas (como óvalos, rectángulos y diamantes) para representar pasos, decisiones y flujos.

7. Pasos para resolver un problema

  1. Entender el problema.
  2. Definir entradas y salidas.
  3. Diseñar el algoritmo.
  4. Probar el algoritmo con casos.
  5. Implementar en un lenguaje de programación.
  6. Depurar y optimizar.

8. ¿Qué es Java?

Java es un lenguaje de programación orientado a objetos, robusto y multiplataforma, utilizado en aplicaciones empresariales, móviles (Android), y web. Su lema es "escribe una vez, ejecuta en cualquier lugar" gracias a la Máquina Virtual de Java (JVM).

¿Qué es la Máquina Virtual de Java (JVM)?

La JVM es un entorno de ejecución que permite correr programas Java en cualquier dispositivo o sistema operativo. Convierte el código Java compilado (bytecode) en instrucciones que el hardware puede entender.

¿Qué es el JDK?

El JDK (Java Development Kit) es un conjunto de herramientas para desarrollar aplicaciones Java, incluyendo el compilador (javac), la JVM, y bibliotecas estándar.

¿Qué es una clase en Java?

Una clase es una plantilla que define las propiedades (atributos) y comportamientos (métodos) de un objeto. Es la base de la programación orientada a objetos en Java.

9. Editores de código e IDEs para Java

Editores de código: Visual Studio Code (con extensiones Java), Sublime Text.

IDEs: IntelliJ IDEA, Eclipse, NetBeans.

Editores con IA: GitHub Copilot, integrado en IDEs como IntelliJ para sugerencias de código.

10. Herramientas NoCode

Plataformas NoCode permiten crear software sin programar: Bubble, Webflow, AppGyver.

11. Ecosistema de Java

El ecosistema de Java incluye bibliotecas, frameworks (como Spring, Hibernate, Quarkus), herramientas de compilación (Maven, Gradle), y una comunidad activa.

12. Frameworks de Java

Un framework es un conjunto de herramientas y bibliotecas que facilita el desarrollo. Ejemplos en Java:

  • Spring: Framework para aplicaciones empresariales, ideal para backend.
  • Hibernate: Framework para mapeo objeto-relacional (ORM).
  • Quarkus: Framework optimizado para aplicaciones nativas en la nube y microservicios.

13. Backend

El backend es la parte del software que maneja la lógica, bases de datos y servidores. Ejemplo: Java Spring, Quarkus.

14. Frontend

El frontend es la interfaz que el usuario ve e interactúa. Usa HTML, CSS y, en algunos casos, Java con frameworks como JavaFX.

15. Protocolo HTTP

HTTP (HyperText Transfer Protocol) es el protocolo para transferir datos en la web. Incluye peticiones (GET, POST) y respuestas (200 OK, 404 Not Found).

16. Servidor web

Un servidor web es un software o hardware que responde a peticiones HTTP, sirviendo contenido como páginas web. Ejemplo: Apache Tomcat, Jetty.

17. Hosting y Dominio

Hosting: Servicio que almacena tu sitio web en servidores. Ejemplo: AWS, Hostinger.

Dominio: La dirección de tu sitio web, como www.ejemplo.com.

18. Fundamentos de Java

Primer programa: Hola Mundo

public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("Hola, Mundo!");
    }
}

Guarda este código en un archivo HolaMundo.java, compílalo con javac HolaMundo.java y ejecútalo con java HolaMundo.

Tipos de datos en Java

  • Primitivos:
    • byte: 8 bits (-128 a 127)
    • short: 16 bits (-32,768 a 32,767)
    • int: 32 bits (-2^31 a 2^31-1)
    • long: 64 bits (-2^63 a 2^63-1)
    • float: 32 bits (decimales)
    • double: 64 bits (decimales precisos)
    • char: Carácter Unicode (16 bits)
    • boolean: true/false
  • Referencia: String, Arrays, Clases

Ejercicios básicos en Java

  1. Variables: Crea un programa que declare variables de diferentes tipos (int, double, String, boolean) e imprima sus valores.
    2. public class Variables {
    public static void main(String[] args) {
        int edad = 25;
        double altura = 1.75;
        String nombre = "Juan";
        boolean esEstudiante = true;
        System.out.println("Edad: " + edad);
        System.out.println("Altura: " + altura);
        System.out.println("Nombre: " + nombre);
        System.out.println("Es estudiante: " + esEstudiante);
    }
}
  2. Ternario: Escribe un programa que use el operador ternario para determinar si un número es par o impar.
    3. public class ParImpar {
    public static void main(String[] args) {
        int numero = 7;
        String resultado = (numero % 2 == 0) ? "Par" : "Impar";
        System.out.println("El número " + numero + " es " + resultado);
    }
}
