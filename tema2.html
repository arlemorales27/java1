Conceptos de Programación Orientada a Objetos

1. ¿Qué es la Programación Orientada a Objetos?

La Programación Orientada a Objetos (OOP) es un paradigma de programación que organiza el código en objetos, que son instancias de clases. Una clase es como un plano que define las propiedades (atributos) y comportamientos (métodos) de un objeto. Los principios clave de OOP incluyen:

  • Encapsulamiento: Ocultar detalles internos y exponer solo lo necesario.
  • Herencia: Una clase puede heredar propiedades y métodos de otra.
  • Polimorfismo: Objetos de diferentes clases pueden ser tratados de manera uniforme.
  • Abstracción: Simplificar problemas modelando solo lo relevante.

Ejemplo: Una clase Suma que realiza la suma de dos números, instanciada desde una clase Main.

public class Suma {
    // Atributos
    private int numero1;
    private int numero2;

    // Constructor
    public Suma(int numero1, int numero2) {
        this.numero1 = numero1;
        this.numero2 = numero2;
    }

    // Método para realizar la suma
    public int calcularSuma() {
        return numero1 + numero2;
    }
}

public class Main {
    public static void main(String[] args) {
        // Crear una instancia de Suma
        Suma operacion = new Suma(5, 3);
        // Llamar al método calcularSuma
        int resultado = operacion.calcularSuma();
        System.out.println("La suma de 5 y 3 es: " + resultado);
    }
}

Explicación: La clase Suma define dos atributos (numero1, numero2) y un método (calcularSuma). En la clase Main, se crea una instancia de Suma con los valores 5 y 3, y se llama al método para obtener el resultado (8).

2. ¿Qué es un Atributo y un Método?

Atributo (o Propiedad): Es una variable que pertenece a una clase y describe una característica del objeto. Por ejemplo, en la clase Suma, numero1 y numero2 son atributos.

Método: Es una función o procedimiento definido dentro de una clase que describe un comportamiento del objeto. Por ejemplo, calcularSuma es un método que realiza la suma de los atributos.

Ejemplo ilustrativo:

public class Persona {
    // Atributos
    String nombre;
    int edad;

    // Método
    void saludar() {
        System.out.println("Hola, mi nombre es " + nombre + " y tengo " + edad + " años.");
    }
}

Explicación: nombre y edad son atributos que almacenan datos, mientras que saludar es un método que realiza una acción usando esos datos.

3. Entrada por Teclado con Scanner

La clase Scanner en Java permite leer entrada del usuario desde el teclado. Se encuentra en el paquete java.util. 

import java.util.Scanner;

public class EntradaScanner {
    public static void main(String[] args) {
        // Crear un objeto Scanner
        Scanner scanner = new Scanner(System.in);

        // Solicitar entrada al usuario
        System.out.print("Ingrese su nombre: ");
        String nombre = scanner.nextLine();

        System.out.print("Ingrese su edad: ");
        int edad = scanner.nextInt();

        // Mostrar los datos ingresados
        System.out.println("Nombre: " + nombre + ", Edad: " + edad);

        // Cerrar el scanner
        scanner.close();
    }
}

Explicación: Se importa Scanner, se crea un objeto para leer datos, se usa nextLine() para cadenas y nextInt() para enteros. Finalmente, se cierra el Scanner para liberar recursos.

4. Entrada con JOptionPane

JOptionPane (del paquete javax.swing) permite crear ventanas emergentes para entrada y salida de datos de forma gráfica. 

import javax.swing.JOptionPane;

public class EntradaJOptionPane {
    public static void main(String[] args) {
        // Solicitar entrada al usuario
        String nombre = JOptionPane.showInputDialog("Ingrese su nombre:");
        String edadStr = JOptionPane.showInputDialog("Ingrese su edad:");

        // Convertir edad a entero
        int edad = Integer.parseInt(edadStr);

        // Mostrar mensaje
        JOptionPane.showMessageDialog(null, "Nombre: " + nombre + ", Edad: " + edad);
    }
}

Explicación: showInputDialog muestra una ventana para ingresar datos, y showMessageDialog muestra un mensaje. La entrada de edad se convierte de cadena a entero usando Integer.parseInt.

5. Conversiones y Manejo de Cadenas

Conversiones con Integer.parseInt: Convierte una cadena que representa un número en un entero. Ejemplo: Integer.parseInt("123") devuelve 123.

Manejo de Cadenas (String): Las cadenas son objetos en Java que permiten operaciones como concatenación, obtener longitud, o extraer subcadenas. 

public class ManejoCadenas {
    public static void main(String[] args) {
        // Conversión de cadena a entero
        String numeroStr = "42";
        int numero = Integer.parseInt(numeroStr);
        System.out.println("Número convertido: " + numero);

        // Operaciones con cadenas
        String texto = "Hola, Mundo!";
        System.out.println("Longitud: " + texto.length());
        System.out.println("Subcadena: " + texto.substring(0, 4));
        System.out.println("Mayúsculas: " + texto.toUpperCase());
        System.out.println("Concatenación: " + texto + " Bienvenido");
    }
}

Explicación: Integer.parseInt convierte una cadena a entero. Métodos como length(), substring(), y toUpperCase() manipulan cadenas.

6. Estructuras de Decisión: if y switch

Estructura if: Evalúa una condición y ejecuta un bloque de código si es verdadera. Puede incluir else if y else. 

public class EjemploIf {
    public static void main(String[] args) {
        int edad = 20;

        if (edad >= 18) {
            System.out.println("Es mayor de edad.");
        } else if (edad >= 13) {
            System.out.println("Es adolescente.");
        } else {
            System.out.println("Es menor de edad.");
        }
    }
}

Estructura switch: Evalúa una variable y ejecuta un bloque según su valor. Ideal para múltiples casos discretos. 

public class EjemploSwitch {
    public static void main(String[] args) {
        int dia = 3;
        String nombreDia;

        switch (dia) {
            case 1:
                nombreDia = "Lunes";
                break;
            case 2:
                nombreDia = "Martes";
                break;
            case 3:
                nombreDia = "Miércoles";
                break;
            default:
                nombreDia = "Día no válido";
                break;
        }
        System.out.println("El día es: " + nombreDia);
    }
}

Explicación: if evalúa condiciones secuencialmente, mientras que switch selecciona un caso específico. break evita que se ejecuten casos posteriores en switch.
